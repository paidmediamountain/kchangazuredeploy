Getting Started

1

Install the Plugin

In PyCharm: Settings → Plugins → Marketplace → search KChang Azure Deploy → Install.

2

Install & Login to Azure CLI

Download Azure CLI, then run az login in your terminal.

3

Configure the Plugin

Go to Settings → Tools → Azure Function Deploy, click Fetch from Azure CLI to auto-fill your subscription, then click Fetch Apps from Azure to pick your Function App.

4

Deploy

Go to Tools → Deploy to Azure Function App, confirm the dialog, and watch the live log in the Azure Deploy panel.
