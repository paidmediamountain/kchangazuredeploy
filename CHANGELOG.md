# Changelog

All notable changes to KChang Azure Deploy will be documented here.

## [1.0.0] — 2026-03-19

### Added
- Azure Explorer left sidebar tree showing all Function Apps with live status
- One-click zip deployment to Azure via Azure CLI
- Start, Stop, Restart Function Apps from the IDE
- Live log streaming in a dedicated tab
- Application Settings editor (view, add, edit, remove)
- Upload local.settings.json to Azure
- Download Azure settings to local.settings.json
- Function browser — see all functions with trigger types
- Copy Function URL with function key
- Create new Azure Function Apps without leaving PyCharm
- Deployment confirmation dialog
- Per-project settings stored in .idea/
- Auto-fetch Subscription ID from Azure CLI
- Auto-fetch Function Apps from Azure CLI
- Freemium model with 30-day free trial via JetBrains Marketplace
